How to Embed a Service-Hosted Video

    
    
        
            

Step 1: Find the Video

            

Go to the video hosting service and find the video you wish to embed.

                     
        
            

Step 2: Get the Embed Code

            

Click on the 'Share' button below the video and select the 'Embed' option.

                     
        
            

Step 3: Copy the Embed Code

            

Copy the HTML embed code provided by the video service.

                     
        
            

Step 4: Paste the Embed Code into Your HTML

            

Open your HTML file and paste the embed code where you want the video to appear.
